# Installation

Clone the repo
```
git clone https://github.com/delltaxa/atlas.git
```

Direct in to the Directory
```
cd atlas/
```

Install packages
```
pip install -r requirements.txt
```

One-Liner
```
git clone https://github.com/delltaxa/atlas.git; cd atlas/; pip install -r requirements.txt
```
